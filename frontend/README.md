# Messages App Frontend

- If you are in root directory run `cd frontend/`.

- Required `node 16.15.1`

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```
