# Messages App Frontend

- If you are in root directory run `cd frontend/`.

- Required `node 16.15.1`

## Project setup
```
yarn
```

### Compiles and hot-reloads for development
```
yarn serve
```

### Compiles and minifies for production
```
yarn build
```

### Lints and fixes files
```
yarn lint
```
